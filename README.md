# consulta-medica-atividade
Sistema de agendamento médico desenvolvido com HTML, CSS e JavaScript. O projeto gerencia fluxos distintos para 3 níveis de acesso: Paciente, Médico e Secretária. Foco na organização de permissões, manipulação de DOM para alternância de perfis e layout intuitivo. Desenvolvido com suporte de IA para otimização da lógica e interface.
